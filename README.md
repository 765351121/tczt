<h1 align="center">tczt</h1>

## Features
- :white_check_mark: **Tech Stack **: vue-cli3 + vue + vue-router + vuex

## Usage

```javascript
$ git clone https://github.com/765351121/tczt.git
$ cd tczt
```

#### Install

```javascript
npm install
```
```javascript
yarn install
```

#### Start
```javascript
npm run serve
```

```javascript
yarn serve
```
Open your browser and visit http://127.0.0.1:8080 , see more at Development.
